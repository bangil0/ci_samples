INSTALLATION INSTRUCTIONS

STEP 1: upload to web host

Upload all files in the ci_membership folder to your web host so just the content inside without the folder.

STEP 2: configuring .htaccess file

Most hosts will allow the usage of a root htaccess file to allow for certain default server routing. We will remove the index.php from the url and set a redirect from www to non-www.

CASE: installation in a subfolder needs the RewriteBase rule. LINE 3 contains "RewriteBase /";
When for example we install in dev/cim3/ then we need to change this to RewriteBase /dev/cim3/.

Rerouting www to non-www (optional):
LINE 5-7.

Removing index.php from URI's
LINE 9-13. Leave as is.

Google web fonts: LINE 15-19 allows certain file types to run smoothly by predeclaring them with AddType.

STEP 5: running the installer on http://yoursite.com/[subfolders]/installer/install.php

This is not (yet) a full installer but merely creates database and admin account based on the config data entered. All the above steps will disappear in with time.

STEP 6: log in and change some vital settings

DEFAULT LOGIN: administrator / 1234

Change the administrator password: go to profile and set a new password for your administrator account.

Change the administrator email account: go to site settings and scroll down to "Administrator e-mail account".

(optional) Enable RecaptchaV2 and set the required keys: go to site settings and scroll to the bottom to enable it and enter this data.
