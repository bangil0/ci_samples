REQUIREMENTS

Needs at least PHP 5.5, using mcrypt and mod_rewrite enabled. Mysql 5.x.

IMPORTANT - errors during account creation, run above root

When you get an error adding or removing members that a certain folder is not found you will need to chmod assets/img/members to 777. Unfortunately this means your server is not properly configured and it would be wise to run the whole installation outside or above root if possible. For more information on how to do this, because this is actually always a good idea, please check this Codeigniter manual entry here.
